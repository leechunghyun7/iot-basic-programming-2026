- 프로그램이란 : 사용자의 요구에 따라 데이터를 처리(표현,이동,저장,로드...)할 수 있는 소프트웨어

- 프로그램을 만드는 작업 : 프로그래밍 (코딩, sw개발,소프트웨어 엔지니어)
